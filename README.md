## 能实现四则运算的计算器
### 功能:
- 1.实现四则运算
- 2.能够检查溢出、格式不正确等异常

### 代码实现
详情代码见上述文件夹中的代码。  

界面设计十分简单，学完Android中的Layout布局控件就可以设计了。  

主要是逻辑功能实现，需要熟练掌握String类的相关API，如subString()、charAt()、contains()、indexOf()、valueOf()等常用方法即可。  

控制运算结果的位数在0~12位之间。
